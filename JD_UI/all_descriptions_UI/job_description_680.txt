Position: React Native Front-End Developer 

Position Description

CGI is seeking a React Native Front-End Developer with expertise in integrating mobile applications with Salesforce.

Your Future Duties And Responsibilities

 Mobile UI/UX Development: Design and develop intuitive, responsive, and visually appealing user interfaces for our React Native mobile application, adhering to best practices and user experience guidelines.
 Salesforce API Integration: Collaborate with backend developers and Salesforce administrators to integrate the mobile application with Salesforce using REST APIs, ensuring smooth data synchronization and real-time updates.
 Data Management: Manage data flow between the mobile application and Salesforce, implementing mechanisms for data validation, synchronization, and error handling to ensure data accuracy and integrity.
 Performance Optimization: Identify performance bottlenecks in the application and optimize code, API calls, and data caching to deliver a smooth and responsive user experience.
 Cross-Functional Collaboration: Work closely with product managers, designers, and backend developers to translate requirements into technical specifications and deliver high-quality solutions.
 Unit Testing: Write unit tests for the React Native components and functionality, ensuring code reliability and maintainability.
 Documentation: Document the codebase, API integrations, and technical decisions to facilitate knowledge sharing and onboarding for other team members.
 Troubleshooting and Bug Fixing: Investigate and resolve issues related to the mobile application's UI, functionality, and Salesforce integration, collaborating with cross-functional teams as necessary.

Continuous Learning: Stay up to date with the latest trends, tools, and best practices in React Native development, Salesforce integration, and mobile application development in general.

Proven experience developing mobile applications using React Native, with a strong portfolio of relevant projects.

Proficiency in JavaScript, ES6+, and React Native libraries and tools

Familiarity with Salesforce's data model, REST APIs, and security mechanisms

Strong written and oral communication skills. Able to document codebase, API integrations, and facilitate knowledge transfer.
