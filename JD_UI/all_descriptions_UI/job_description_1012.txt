About The Opportunity

Aspen Square Management, one of the nation’s largest privately held real estate investment and property management firms with over $2 Billion in assets, is seeking a creative, highly skilled  Front-End Application Developer to join our team. The ideal candidate will have a strong background in front-end development and be proficient in technologies such as React, React Native, JavaScript, Sass, CSS, and Node.js. You will be responsible for designing and developing user-friendly, responsive, and visually appealing web and mobile applications that deliver exceptional user experiences.

Responsibilities

 Collaborate with cross-functional teams, including designers and back-end developers, to translate design mockups and wireframes into functional and visually appealing front-end applications.
 Develop and maintain responsive web and mobile applications using React and React Native.
 Write clean, efficient, and maintainable code following best practices and coding standards.
 Implement and optimize user interfaces for maximum performance and usability.
 Work closely with UI/UX designers to ensure the design vision is faithfully translated into the final product.
 Debug and troubleshoot issues to ensure smooth application performance across various browsers and devices.
 Stay up-to-date with the latest front-end development trends, tools, and technologies.
 Collaborate with the back-end development team to integrate front-end components with server-side logic using Node.js.
 Contribute to the development of reusable components and libraries to streamline future development efforts.
 Participate in code reviews to maintain code quality and ensure adherence to coding standards.

Requirements

 Bachelor's degree in Computer Science, Information Technology, or a related field (or equivalent work experience). 
 Proven experience as a Front-End Developer with a strong portfolio of web and mobile applications. 
 Proficiency in React, React Native, JavaScript, HTML5, and CSS3. 
 Experience with Sass for CSS preprocessing. 
 Familiarity with Node.js and server-side integration. 
 Knowledge of RESTful APIs and asynchronous programming. 
 Strong problem-solving and debugging skills. 
 Excellent collaboration and communication skills. 
 Attention to detail and a commitment to delivering high-quality work. 
 Ability to work both independently and as part of a team. 
 Strong organizational skills and the ability to manage multiple tasks and projects. 

Preferred Qualifications

 Experience with front-end build tools like Webpack, Babel, Storybook, and npm. 
 Knowledge of version control systems (e.g., Git). 
 Experience with testing frameworks and methodologies (e.g., Jest, Enzyme). 
 Familiarity with responsive design principles. 
 Experience with cross-platform mobile development. 
 Understanding of web security best practices.
