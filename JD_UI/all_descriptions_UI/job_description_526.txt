DMI, LLC is in search of a SAP Commerce Cloud / Hybris UI Developer to aid our west coast based retail client in their transition to the latest version of SAP Commerce Cloud. This position is entirely remote supporting PST hours.

 Proficiency in HTML, JS, Bootstrap, Ajax, jQuery, CSS & Less.

Hands-on experience in Spartacus as front end framework for SAP commerce cloud
