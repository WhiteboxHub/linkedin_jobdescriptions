Inviting applications for the role of Lead Consultant, Boomi Integration Developer!

Seeking a highly skilled and experienced Boomi Integration Developer with a primary focus on developing API services using the Boomi API management platform. The ideal candidate should possess a minimum of hands-on experience in Boomi integration, with a strong emphasis on API development and management. The candidate should also have a deep understanding of API security, particularly OAuth, and adhere to best design standards for API development.

Responsibilities

 API Development: Develop, design, and maintain high-quality API services using the Dell Boomi platform to enable seamless integration between various systems, applications, and data sources.
 Boomi API Management: Configure and manage API endpoints, policies, and security settings within the Boomi API management platform to ensure efficient and secure API access.
 Integration Workflows: Create and manage integration workflows, transformations, and error handling processes to meet business requirements.
 API Security: Implement and maintain API security measures, including OAuth authentication and authorization, to protect sensitive data and ensure compliance with security standards.
 Best Practices: Follow industry best practices and design standards for REST API development, ensuring scalability, performance, and maintainability of APIs.
 Collaboration: Collaborate with cross-functional teams, including architects, developers, and business stakeholders, to gather integration requirements and provide technical expertise .
 Documentation: Maintain detailed documentation of API services, configurations, and integration workflows to support troubleshooting, monitoring, and future enhancements.
 Performance Optimization: Identify and resolve performance bottlenecks and optimize integration processes for maximum efficiency.
 Testing and Validation: Conduct thorough testing, debugging, and validation of API services to ensure they meet functional and quality standards.

Qualifications we seek in you!

Minimum Qualifications

 Hands-on experience with Dell Boomi integration platform.
 Proficiency in developing and managing API services using Boomi API management.
 Strong knowledge of API security principles, especially OAuth.
 Familiarity with integration design patterns and best practices.
 Excellent problem-solving skills and attention to detail.
 Effective communication and collaboration skills.
 Ability to work in a fast-paced, dynamic environment.
